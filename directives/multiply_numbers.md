# Directive: Multiply Numbers

**Goal**: Multiply two numbers provided as input.

## Inputs
- `number1`: The first number
- `number2`: The second number

## Tools
- `execution/multiply_numbers.py`

## Process
1.  Read the input numbers.
2.  Run the python script `execution/multiply_numbers.py` with the numbers as arguments.
3.  Output the result.

## Output
- The product of the two numbers.
